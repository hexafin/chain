# Hexa chain

## Install

` yarn `

## Deploying

` firebase deploy `

## Directory

```
    chain
    ├── .circleci                # continuous integration pipelines
    ├── archive                  # old (unused code)
    ├── functions                # holds functions triggered by HTTP requests and database changes
    ├── node_modules             
    ├── .firebaserc              # project name
    ├── firebase.json            # defines rules file
    ├── firebase.rules           # database access rules
    ├── firebase.indexes.json    # database indexing order
    ├── yarn.lock       
    └── README.md           
```
